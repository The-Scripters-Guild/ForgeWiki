The Test Node is a component designed to facilitate signal routing between input and output channels via the main bus.

## Signal Routing

The Test Node connects to the main bus and handles the routing of signals between its input and output channels.

## Chaining and Connectivity

The output of the Test Node can be connected to any downstream node.

### Multiple Node Chains

For more complex setups, multiple Test Nodes can be connected in a chain.
